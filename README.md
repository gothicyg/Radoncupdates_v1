# Radiation Oncology Signals

A curated platform for clinically relevant radiation oncology literature.

## Workflow

- Automated ingestion from oncology literature sources
- AI-assisted classification and triage
- Editorial review and approval
- Public presentation of selected articles

## Current Features

- Disease-specific categorization
- PubMed integration
- Editorial approval workflow
- Weekly curated updates

## Technology

- Next.js
- Supabase
- Gemini AI
- PubMed APIs
